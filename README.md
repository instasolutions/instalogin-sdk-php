# Instalogin PHP SDK

A simple PHP wrapper for the Instalogin API. 

## Prerequisites

To successfully initialize the PHP client, you'll need an API key and API secret from Instalogin.

```ini
# Example .env Data

INSTALOGIN_KEY=...
INSTALOGIN_SECRET=...
```
